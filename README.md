# ToolKit Pro — Built for Digital Heroes

A free, all-in-one web toolkit with 9 powerful tools, built entirely with HTML, CSS, and JavaScript.

## 🔧 Tools Included

| Tool | Description |
|------|-------------|
| GST Calculator | Add/remove GST with CGST/SGST breakdown |
| EMI / Loan Calculator | Monthly EMI, total interest, total payment |
| Image Compressor | Client-side compression with quality control |
| QR Code Generator | Custom color & size QR codes, downloadable |
| Unit / Currency Converter | 7 categories including currency (INR base) |
| Split the Bill | Tip, discount, per-person split |
| Age Calculator | Years/months/days + days to next birthday |
| Word & Character Counter | Words, chars, sentences, read time |
| Color Palette Generator | Analogous, complementary, triadic, shades |
